Security settings

You have to "whitelist" this page to be able to run the QuBE program. You can find more information about that on this page.

In short, you have to follow these steps:

  • Start the java settings console (Windows Start Menu > Programs > Java > Configure Java)
  • Go to the "Security" tab
  • At the bottom, in the "Exception Site List" section, click "Edit Site List..."
  • Click the "Add" button and add 
    http://www.stellarjackpot.com/
  • Click ok and close the console.
You will still get scary warnings that the app is potentially dangerous, there's unfortunately nothing we can do about that at this time.
